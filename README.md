
# 새싹블로그
- 로그인, 회원가입기능을 구현해보았습니다.
- 게시글 작성/수정/삭제 - CRUD
- 댓글 작성/수정/삭제  - CRUD
- jwt토큰을 사용하여 사용자를 구분하였습니다.
- 프로젝트의 전반적인 설계를 MVVM패턴으로 진행하고 일부분은 MVC패턴을 사용했습니다.
- URLSession을 사용하여 네트워크 통신을 진행했습니다.

> 오픈소스
- SnapKit
- JJFloatingActionButton
