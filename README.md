# 프로젝트
위치기반 사람 찾기 프로젝트

> 프로젝트 진행방식

- 진행기간 : 1/17(월) ~ 2/18(금) 
- 주중 팀원(4인)들과 진행상황 및 이슈 공유하면서 개인프로젝트로 진행
- UI디자인 및 서버는 제공받은 후 클라이언트 ios앱 개발진행
